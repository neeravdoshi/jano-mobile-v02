## Design Context

### Users
Doctors using a mobile companion app during their clinical day. Context: hospital/clinic environment, often time-pressured, glancing at the app between patients. Jobs: track patients, view labs, manage prescriptions and schedule.

### Brand Personality
Calm · Clinical · Precise. The interface should feel like a trusted colleague — confident and reliable, never noisy or over-designed. It reduces cognitive load; it doesn't add to it.

### Aesthetic Direction
- **Theme**: Light mode. Doctors use this in bright clinical settings, on white coat pocket glances.
- **Palette**: Jano Crimson (`#E54B4B`) as the single accent — used sparingly for brand, primary actions, and urgent states. Warm charcoal text on warm off-white surfaces. No gradient fills, no glass morphism, no decorative shadows.
- **Typography**: Figtree — single typeface. Display scale (24/32/48px) for screen heroes; UI scale (12–20px) for all list/card/label content. No hard-coded sizes — always use `type-*` classes.
- **Density**: Moderate — not as dense as an EMR, not as airy as a consumer wellness app. Data should be readable at a glance.
- **Anti-references**: Generic hospital software (grey, clunky, form-heavy), consumer health/fitness apps (gamified, bright, friendly-to-a-fault), startup dashboards (big metrics, sparklines, gradient cards).

### Design Principles
1. **Space communicates hierarchy.** Generous gaps between sections, tight grouping within them. No uniform padding everywhere.
2. **One accent.** Crimson appears only on brand identity, primary actions, and urgent/critical states. Overuse kills its power.
3. **Tokens only.** Every color is a `var(--color-*)` alias. Every size is a `type-*` class. No hardcoded hex or px in components.
4. **Rhythm over decoration.** Visual interest comes from well-proportioned spacing and type scale — not shadows, borders, or gradient overlays.
5. **Mobile-first, phone-sized.** The shell is 390×844px. Every layout decision should look right at that width, with the MobileShell chrome in mind.
